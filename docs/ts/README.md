## TypeScript

> TypeScript 是 JavaScript 的一个超集，主要提供了类型系统和对 ES6 的支持.

[TypeScript 官网](https://www.tslang.cn/)

TypeScript 的优点：

1. 能在开发过程中更快的发现潜在问题。
2. 对编辑器更友好的代码提示功能。
3. 代码语义更清晰易懂。
